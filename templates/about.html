About this project

A teaching-quality information-retrieval engine built from scratch, covering the canonical NLP and IR pipeline used by classical search systems before the rise of neural retrievers.

System

What it does

①

Indexing

Reads a text or JSON corpus, normalizes tokens, and builds an inverted index with positions and per-document statistics.

②

Ranking

Scores candidate documents with TF-IDF (cosine) or Okapi BM25 with tunable k1 and b.

③

Querying

Parses bare terms, +must / -must-not, quoted phrases, and boolean operators.

④

Presentation

Picks the best window, highlights matches, and offers a spell suggestion when no documents match.

JSON

API endpoints

GET /api/search?q=machine+learning&model=bm25&k=10
GET /api/stats

Both routes return JSON; the search endpoint also includes the Damerau-Levenshtein spell suggestion when relevant.
