Capabilities

What's inside

⌁

NLP preprocessing

Tokenization, case folding, stop-word removal, and a compact Porter stemmer.

≡

Inverted index

Postings carry term frequencies and positions, enabling phrase queries.

∑

TF-IDF + BM25

Two interchangeable rankers; tune k1/b for BM25 in config.py.

¶

Smart snippets

Best-window selection with highlighted query terms in the original text.

✎

Spell suggestions

Damerau-Levenshtein matching against the index vocabulary.

◐

Dual front-ends

Argparse CLI plus this Flask web app. JSON API at /api/search.

Pipeline

How a query flows

  1. Parse

    The query parser extracts phrases, +must, -must-not, and boolean ops.

  2. Normalize

    Tokens are lowercased, stop-filtered, and Porter-stemmed.

  3. Score

    BM25 (default) or TF-IDF scores every candidate document.

  4. Render

    Top-k results are filtered, snipped, and highlighted for display.

Run it

Quickstart

$ pip install -r requirements.txt
$ python scripts/build_index.py
$ python app.py      # open http://127.0.0.1:5000
$ python cli.py "machine learning"
